# Dostępne Światy Gazebo

## Uruchamianie

```bash
# VTOL (knr_tiltrotor)
./scripts/run_px4_vtol_sitl.sh <nazwa_świata>

# Quadrotor (x500)
./scripts/run_px4_quad_sitl.sh <nazwa_świata>
```

Domyślny świat: `aruco`
